## **项目简介**
这是一个基于python request库做的爬虫程序，所有使用铃铛教育的学校学生均可以用这个项目快速，自动，高效，快捷，智能地完成学校作业。这个项目也使用了豆包，deepseek等的api可以使用ai模型完成作业，也支持自动生成录音完成阅读作业。
## **食用方法（名字是抽象）**
